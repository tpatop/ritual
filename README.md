# Описание

Этот репозиторий предназначен для упрощения управления нодой **RITUAL** и улучшения пользовательского опыта. Он позволяет быстро настроить и запустить необходимые процессы с минимальными усилиями.

### Основные возможности:
- Автоматизация запуска и настройки ноды
- Удобный интерфейс для управления нодой

![Скриншот интерфейса](https://github.com/user-attachments/assets/e664b2a6-b142-4e06-aefc-69c1c79d0292)

### Запуск скрипта

Если вы хотите запустить скрипт без сохранения на сервере, используйте следующую команду:

```bash
bash <(curl -s https://raw.githubusercontent.com/tpatop/ritual/main/ritual-ru.sh)
```
